SSAD Project
Butterfly Impact is a game to learn about Software Engineering. The game consists of of levels and sublevels, each representing different phases of the life cycle of Software Engineering, for students to do quizzes and learn in a fun and engaging way. The game supports both single player and multiplayer modes, and also allows users to create their own custom lobbies. Students can unlock levels as they play and keep track of their scores on the leaderboad. Teachers have admin access and can create new assignments, share them with students on social media, as well as monitor student's progress using a summary report generated for all students.
